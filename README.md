# LemonF

LemonF is a parser and serializer for data to be easily read and modifiable for the user.

Featues
-

- Bodys
- Bodys with tags
- Variables (messages)
- Serialization
--
- Integer
- Float
- Double
- Char
- Const char*
- STD:String
- Classes (more info in the wiki)
